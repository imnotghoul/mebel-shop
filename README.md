# mebel-shop

Landing page for a renovation and interior services portfolio project.

## Files

- `index.html` - page markup
- `styles.css` - layout, responsive styles, animations
- `script.js` - scroll reveal, counters, form behavior
- `assets/` - generated visual assets

## Run locally

Open `index.html` in a browser or start a simple static server:

```bash
python -m http.server 5173
```

Then open `http://127.0.0.1:5173/index.html`.
