# mebel-shop

Лендинг для портфолио: сайт студии ремонта и интерьерных услуг.

## Что внутри

- `index.html` - разметка страницы
- `styles.css` - стили, адаптивность и анимации
- `script.js` - анимации при скролле, счетчики и поведение формы
- `assets/` - изображения для сайта

## Локальный запуск

Можно открыть `index.html` напрямую в браузере или запустить простой локальный сервер:

```bash
python -m http.server 5173
```

После этого откройте `http://127.0.0.1:5173/index.html`.
